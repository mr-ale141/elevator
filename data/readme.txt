Файлы HGT могут быть прочитаны и преобразованы в другие форматы DEM с помощью 3dem.

Файл HGT занимает площадь 1°x1°. Его юго-западный угол можно определить по названию файла: например, n51e002.hgt охватывает область между N 51° 2° Восточной долготы и N 52° 3° восточной долготы, а s14w077.hgt охватывает область от 14° 77° южной широты до 13° 76° южной широты. Размер файла зависит от разрешения. Если это значение равно 1", то в нем 3601 строка по 3601 ячейке в каждой; если оно равно 3", то в нем 1201 строка по 1201 ячейке в каждой. Строки расположены как текст на странице, начиная с самой северной строки, причем каждая строка читается с запада на восток. Каждая ячейка содержит два байта, а высота над уровнем моря в этой ячейке равна 256*(1-й байт) + (2-й байт). Из этого следует, что файл формата HGT размером 3 дюйма имеет длину 2 x 1201 x 1201. Ячейки формата SRTM размером 3 дюйма рассчитываются путем вычисления среднего значения для ячеек размером 1 дюйм и их восьми соседей. Из этого следует, что самая высокая локальная точка, скорее всего, будет выше, чем самая высокая ячейка SRTM3". Разница должна варьироваться в зависимости от крутизны местного рельефа.
